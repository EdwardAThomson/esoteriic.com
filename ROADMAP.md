# Roadmap — esoteriic.com

_Status: active · updated 2026-05-30_

Personal blog — a Node.js static-site generator that builds 250+ markdown posts
into a categorized, themed site, plus tooling to import legacy content.

## Shipped

- [x] Static blog generator from markdown posts with front-matter metadata
- [x] Homepage with featured post + post cards grouped by year
- [x] Category index page and per-category archive pages
- [x] Auto table of contents (3+ headings) with scroll-spy active highlighting
- [x] Dark mode toggle (system detection + localStorage persistence)
- [x] Mobile-responsive navigation with hamburger menu
- [x] Code syntax highlighting with copy-to-clipboard buttons
- [x] Lazy image loading
- [x] Anonymity series — multi-chapter layout with floating sidebar navigation
- [x] 300+ legacy URL redirects from the WordPress migration
- [x] Incremental build caching for faster rebuilds
- [x] Medium / Steemit / Wayback import & conversion tooling
